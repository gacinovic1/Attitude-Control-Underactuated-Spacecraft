# Attitude-Control-Underactuated-Spacecraft
Some theoretical results on the attitude stabilization around equilibria of a rigid spacecraft with at least an unactuated thruster are explored.
The main reference is:
C. I. Byrnes and A. Isidori, ”On the Attitude Stabilization of Rigid Spacecraft,” Automatica, vol. 27, no. 1, pp. 87-95, 1991.
